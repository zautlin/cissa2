DETAILED FINDINGS
================================================================================

1. FIXED Approach (Lines 833-840)
   ─────────────────────────────

   Input:
     - spot_betas DataFrame
     - ticker_avg (pre-computed at line 794)
     - beta_rounding parameter (e.g., 0.1)

   Logic:
     FOR EACH ROW:
       beta = ROUND(ticker_avg / beta_rounding, 0) * beta_rounding

   Output:
     DataFrame with same beta value for all fiscal years of a ticker

   Example (BHP AU Equity):
     ticker_avg = 1.05
     β for 2000 = ROUND(1.05/0.1, 0)*0.1 = 1.0
     β for 2001 = ROUND(1.05/0.1, 0)*0.1 = 1.0 ← SAME
     β for 2023 = ROUND(1.05/0.1, 0)*0.1 = 1.0 ← SAME


2. Floating Approach (Lines 841-881)
   ──────────────────────────────────

   Input:
     - spot_betas DataFrame sorted chronologically
     - beta_rounding parameter

   Logic:
     FOR EACH TICKER (chronologically):
       FOR EACH FISCAL YEAR (cumulative window):
         cumulative_avg = MEAN(spot_slope[inception:current_year])
         floating_beta[year] = cumulative_avg
     
     FOR EACH ROW:
       beta = ROUND(floating_beta / beta_rounding, 0) * beta_rounding

   Output:
     DataFrame with different beta value for EACH fiscal year

   Example (BHP AU Equity inception 2002):
     2002: spot_slope=1.1  → floating_beta=1.1   → β=1.1
     2003: spot_slope=1.2  → floating_beta=1.15  → β=1.1 or 1.2
     2004: spot_slope=1.0  → floating_beta=1.1   → β=1.1
     ...
     2020: spot_slope=1.1  → floating_beta≈1.11  → β=1.1
